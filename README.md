# Digital Forensics Labs

This repository documents hands-on digital forensics exercises focused on evidence acquisition, analysis, and interpretation.

## Scope
- Disk and file system analysis
- Memory forensics (introductory)
- Log and timeline analysis
- Incident reconstruction

## Tools
- Autopsy
- Volatility
- Plaso / log2timeline
- Native OS utilities

## Content
- Case-based labs
- Evidence handling notes
- Analysis reports
- Findings and conclusions

## Disclaimer
All cases are simulated and do not involve real or sensitive data.
